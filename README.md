# React + Vite

# Instruction for installation

create a project folder 
type these commands in terminal
1. npm create vite@latest
2. cd vite-project 
3. npm i
4. npm run dev

# install these dependencies:
"react-redux": "^9.1.2",
    "react-router-dom": "^6.27.0"
